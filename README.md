# CampusSwap 🛒

A simplified, production-grade marketplace for campus products.

## ✨ Features
- **Modern UI**: Dark mode, aesthetic design with Tailwind CSS.
- **Mock API**: Local simulated API with search, filter, and pagination support.
- **State Management**: Data fetching powered by **TanStack Query (React Query)**.
- **Routing**: Seamless navigation with **React Router v7**.
- **SOLID Principles**: Clean code structure with decoupled components and custom hooks.

## 🚀 Tech Stack
- React 19 (Vite)
- TanStack Query v5
- React Router v7
- Tailwind CSS v4
- TypeScript

## 🛠️ Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

## 📂 Project Structure
```text
src/
├── components/   # Reusable UI components
├── constants/    # Centralized configuration
├── hooks/        # Custom React & Query hooks
├── layouts/      # Application layouts (Core Structure)
├── pages/        # Application views (Home, 404)
├── services/     # Fake API calls and interaction logic
└── types/        # TypeScript interfaces and types
```
